Инструкция по запуску и подключению к серверу / созданию сервера:

В папке bin->Debug запустить .exe файл
В появившимся окне ввести IP сервера или создать свой
Чтобы узнать IP сервера, на ПК, который выступает в роли сервера, открыть командную строку (cmd.exe), ввести ipconfig. На ПК, который выступает в роли клиента ввести IPv4-адрес сервера.
Чтобы упростить процесс, можно запустить exe файл на одном ПК два раза, в одном окне создать сервер, а в другом прописать localhost и нажать на кнопку Подключиться.
